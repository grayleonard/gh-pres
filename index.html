###what is git? created by linus torvalds version control system both a client and a server - working directory functions as full repository scalable - individual to company-wide download at: http://git-scm.com/downloads
###what is Github ( http://github.com )? also created by linus torvalds ![](./Torvalds-FBomb.jpg) free remote repositories companies use it to 'give back to the community' developers use it to publish and work on projects students use it to get a job (and work on class projects) lots of cool stuff that isn't source code can be found on it
lots of informational repos: * https://github.com/vhf/free-programming-books * https://github.com/kbandla/APTnotes - "Various public documents, whitepapers and articles about APT campaigns" * https://github.com/NARKOZ/guides - "Design and development guides" * https://github.com/PharkMillups/beautiful-docs - "Pointers to useful, well-written, and otherwise beautiful documentation." http://gist.github.com - for code snippets A lot of programs are also completely hosted on github, raw files can be directly linked
